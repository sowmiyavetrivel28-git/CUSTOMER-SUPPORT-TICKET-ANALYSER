# Customer Support Ticket Analyser

## Overview

The **Customer Support Ticket Analyser** is a Python project that simulates a customer support ticket management system. It stores customer tickets, allows users to add new tickets, cleans issue descriptions, and generates useful insights through text and priority analysis.

---

## Features

* Display preloaded customer support tickets
* Add new tickets with auto-increment ticket numbers
* Validate ticket priority (High, Medium, Low)
* Clean issue descriptions by:

  * Removing punctuation
  * Converting text to lowercase
  * Removing extra spaces
  * Replacing shorthand words
* Count tickets containing specific keywords
* Analyze ticket priorities
* Find the ticket with the longest issue description
* Extract unique words from all issue descriptions

## Analytics & Insights

### Descriptive Analytics (What happened?)

* Displayed all customer support tickets.
* Counted High, Medium, and Low priority tickets.
* Counted tickets containing keywords such as **good**, **poor**, **slow**, and **excellent**.
* Identified the ticket with the longest issue description.
* Extracted and displayed unique words from all issue descriptions.

### Diagnostic Analytics (Why did it happen?)

* Analyzed recurring keywords to identify common customer issues.
* Compared ticket priorities to understand the severity of reported problems.
* Used cleaned issue descriptions to identify patterns in customer feedback.

### Predictive Analytics (What might happen?)

* Frequently occurring keywords like **slow** and **poor** may indicate future service-related complaints.
* A higher number of High-priority tickets may suggest an increase in urgent support requests.

### Prescriptive Analytics (What should be done?)

* Prioritize High-priority tickets for faster resolution.
* Improve services related to frequently reported issues.
* Monitor recurring keywords to identify problems early.
* Use customer feedback to improve support quality and response time.
